## Overview
Number Quest is a simple Python mini-game designed using only the concepts learned in class:
- Loops
- Ranges
- Lists
- Functions
- Conditionals (if/elif/else)
- Input/Output
- Random numbers
- Basic math

The project contains **three mini-challenges**:
1. **Guess The Number**  
2. **Range Challenge**  
3. **Even or Odd Detector**
